## Software Arduino IRE 

Programa donde escribes y subes el código

Lenguaje parecido a C/C++

Se conecta al Arduino por USB

**Conexión**

USB → comunicación serial (envía y recibe datos entre computador y Arduino)


**ArduinoLEDMatrix**

Librería para controlar matrices de LEDs

Permite mostrar texto, animaciones, etc.


**Tiempo en Arduino**

Se mide en milisegundos
Importante para delays y temporización


## Tips rápidos

Ctrl + A → seleccionar todo

Revisar siempre el puerto y placa en el IDE antes de subir código

Si no funciona → probar cable/puerto
