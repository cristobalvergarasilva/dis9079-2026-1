### Qué es API? 
Una API (Application Programming Interface) es un puente que permite que dos sistemas se comuniquen entre sí.
En el caso de Adafruit IO, su API permite que tu microcontrolador (Raspberry, Arduino, etc.) envíe y reciba datos desde la nube de Adafruit usando reglas definidas.
